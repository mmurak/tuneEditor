Special key-sequences for Text area

keyDescription
←Move cursor one character to the left.
→Move cursor one character to the right.
shift+←Move auxiliary cursor to the left to select a range.
shift+→Move auxiliary cursor to the right to select a range.
ctrl+qMove cursor to the beginning of the line.
ctrl+wMove cursor to the end of the line.
(This doesn't work with Windows, so use ctrl+shift+Q)
ctrl+shift+QMove cursor to the end of the line.
ctrl+shift+←Extend the selection to the head of the closest word on the left.
ctrl+shift+→Extend the selection to just before the nearest word on the right.
delete (Backspace)Delete text to the left of the cursor. (Or delete the selected region.)
delete[x]> (Delete)Delete text to the right of the cursor. (Or delete the selected region.)
cmd+cCopy the selection to the internal and OS buffer.[*1]
ctrl+cditto
cmd+xCut the selection and copy it to the internal and OS buffer.[*1]
ctrl+xditto
cmd+vPaste the contents from the internal buffer.[*1]
ctrl+vditto
cmd+aSelect all text.
ctrl+aditto

[*1] Due to the current status of JavaScript standards, the behaviour of the cut/copy & paste function (especially paste function) varies from browser to browser. For this reason, this application currently employs internal clipboard, which means the application can't interact with other applications.

Special key-sequences for Graphic area

keyDescription
shift+click on Tone-button-matrixSuppress the automatic focus-forward function.
cmd+shift+←Decrease the width of the tail of the tonal nucleus.
ctrl+←ditto (works with Windows)
cmd+shift+→Increase the width of the tail of the tonal nucleus.
ctrl+→ditto (works with Windows)
cmd+shift+↑Increase the height of the tail of the tonal nucleus.
ctrl+↑ditto (works with Windows)
cmd+shift+↓Decrease the height of the tail of the tonal nucleus.
ctrl+↓ditto (works with Windows)
↑Raise the height of the tail end of the tonal nucleus.
↓Lower the height of the tail end of the tonal nucleus.
cmd+zSelect the tone note immediately to the left.[*2]
ctrl+zditto
cmd+shift+ZSelect the tone note immediately to the right.[*2]
ctrl+shift+Zditto
cmd+kAdjust the height of the graphic area. (between 97 and 199) [*3]
ctrl+shift+kditto (for Windows)

[*2] In this web app, when you click on a Tone-button-matrix (one of the buttons that specifies the height and stress of a tone), the focus automatically moves to the next tone. This means that if you accidentally click an inappropriate button, you will have to re-select the previous tone every time. I know that the best way to solve this problem is to implement the Undo function, but I'm still working on the specifications (^^;), so I decided to implement a "pseudo-Undo" which makes focus move 1 position to the left by pressing Ctrl+Z.

[*3] This feature was implemented at the suggestions of Prof. Yoshiki Nagase. I would like to thank Prof. Nagase, and Mr. Takeo Kubo for passing on the suggestions.
